# Selfhosting

Install [Node.js](https://nodejs.org/) and generate a bot token

Install the dependencies:

```bash
npm install
```

Create a `.env` file in the root directory and add your bot token:

```env
FLUXER_BOT_TOKEN=your_bot_token_here
```

Start the bot with:

```bash
node bot.js
```
